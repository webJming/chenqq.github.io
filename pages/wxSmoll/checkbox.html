创建

组件位置：file->wxSmoll->checkbox

将文件夹checkbox复制到components中

调用

在调用的xx.json或app.json中引入组件：

"usingComponents": { "ysy-checkbox": "/components/checkbox/checkbox" }

页面调用：

< ysy-checkbox radioItems="{{radioItems}}" bind:change="subData" > < / ysy-checkbox >

属性

属性名 类型 说明
radioItems object 生成题目需要传入的数据，具体看上面注释
answer boolean 每做完一题是否马上显示是否正确

事件

事件名 说明 回调参数
change 每改变选项是触发 对象obj，详见“change回调中属性”

change回调中属性

属性名 类型 说明
value String 选中的项
ok boolean 是否正确
data object 该提的数据：如id,图片等
